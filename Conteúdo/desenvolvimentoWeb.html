Objetos de Estudo
Animações com CSS e JavaScript

Para tornar a aplicação web mais atrativa e com um diferêncial, trazendo uma narrativa visual dinâmica e interativa que seja atrativa para o usuário.

Banco de Dados

Um armazenamento muitas vezes é util em sites, para dizer a conta que está logada, se essa conta está cadastrada, além de poder armazenar informações para o usuário, por exemplo: um site de leitura que indica os livros que o usuário iniciou, e o ultímo capítulo lido.

Back-End

Para diversas funcionalidades, o uso de scripts é indispensável ou no minimo facilita a sua execução, um exemplo é o modo escuro e claro, dentre várias outras.

Objetos de Pesquisa
Responsívidade

Fazer aplicações web funcionarem tanto para computadores, quanto tablets e celulares, é uma funcionalidade indispensável para uma boa aplicação web, sendo multiplataforma facilita o acesso para diversos usuários, trazendo mais visualização para a página.

Cores e formas

Cores e formas trazem diversos significados, entendê-los é essencial para deixar a aplicação mais intuitiva. Cores como vermelho e amarelo sendo um exemplo disso sendo frequentemente associadas a fome e comida. Já formas retângulares sendo associadas a organização.

Prototipagem

Já é algo que estudo, veja em Design Gráfico, mas é essencial para a Organização de uma página web.
