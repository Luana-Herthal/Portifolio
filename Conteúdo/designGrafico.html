Objetos de Estudo
Cores e Formas

Cores e formas trazem diversos significados, entendê-los é essencial para deixar a aplicação mais intuitiva. Cores como vermelho e amarelo sendo um exemplo disso sendo frequentemente associadas a fome e comida. Já formas retângulares sendo associadas a organização.

Grids

Para uma melhor organização nos elementos trazendo limites claros em forma de grade, além de estudar os Grids de outras páginas famosas ajuda a entender posiçoes que usuários já estão acostumados.

Estruturação

Posicionamento de informações, buscar estruturações conhecidas para facilitar o entendimento dos usuários, além de tornar mais intuitivo para os usuários.

Objetos de Pesquisa
Responsívidade

Fazer protótipos para computadores, tablets e celulares, é uma prática para o entendimento das dimensões dos camponentes em diversos tamanhos de tela. Além de facilitar na hora do desenvolvimento já com as dimensões trabalhadas e prontas.

Organização de Componentes

Trazer posições coerentes e/ou conhecidas facilita o manuseio para o usuário. Menus no topo ou na lateral direita, textos com imagens do lado, são algumas práticas de organização conhecidas.

Mercado

Entender os designs disponíveis no mercado para estudo, entendendo os designs que são mais apreciados para entender o que é valorizado, além de estudar o que falta no mercado, trazendo novidades e inovações.
